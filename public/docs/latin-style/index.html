Latin Style

Replacements

In general, we follow the medieval manuscript tradition. Thus

  • Ut → Vt, ut → vt
  • J → I, j → i (ij → ii)

When text/handwriting recognition has been used, certain errors are frequent, and can be easily corrected with find and replace:

  • babe → habe
